Adaption Chest X-Ray Pneumonia Labels

**A labeled chest X-ray dataset for automated pneumonia detection.**

Overview

This repository documents the *Adaption Chest X-Ray Pneumonia Labels* dataset — a curated dataset of chest X-ray images with pneumonia/normal labels designed for machine learning and deep learning research.

The dataset enables binary classification between:
- **Pneumonia**
- **Normal**
